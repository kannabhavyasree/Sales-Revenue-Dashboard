# 📊 Sales & Revenue Analysis Dashboard

## 📌 Project Overview

The Sales & Revenue Analysis Dashboard is designed to analyze sales performance, revenue trends, and business KPIs using interactive visualizations. It helps businesses make data-driven decisions by providing insights into sales, profit, customer behavior, and top-performing products.

---

## 🚀 Features

- Import data from Excel or CSV files
- Interactive dashboard with filters and slicers
- Total Sales KPI
- Total Revenue KPI
- Total Profit KPI
- Monthly Sales Trend
- Top Performing Products
- Category-wise Sales Analysis
- Region-wise Sales Analysis
- Customer-wise Sales Analysis

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Microsoft Excel / CSV
- DAX
- Power Query

## ▶️ How to Use

1. Clone or download this repository.
2. Open `Sales_Revenue_Dashboard.pbix` in Microsoft Power BI Desktop.
3. Refresh the data if required.
4. Explore the dashboard using filters and slicers.
